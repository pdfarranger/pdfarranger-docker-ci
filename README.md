# pdfarranger-docker-ci

This repository contains the Dockerfiles used to build docker images for pdfarranger:

* https://hub.docker.com/r/jeromerobert/pdfarranger-docker-ci (ci folder) is used by [Github Actions](https://github.com/jeromerobert/pdfarranger/actions)
* https://github.com/orgs/pdfarranger/packages/container/package/wine-mingw64 is used to create the [Windows .zip and .msi release files](https://github.com/pdfarranger/pdfarranger/releases).
